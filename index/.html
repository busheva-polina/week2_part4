Deployment Notes

To deploy this app on GitHub Pages:

  1. Create a public GitHub repository
  2. Commit index.html and app.js files
  3. Enable GitHub Pages in repository settings (Source: main branch, root folder)
  4. Test the deployed URL (e.g., https://yourusername.github.io/repositoryname)
